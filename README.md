# ResizeBorder v6.0.0

Angular v6

(demo)[https://liangshen001.github.io/resize-border/]

## Usage

`import {ResizeBorderModule} from '../../src/resize-border.module';`

`<div resizeBorder
      [resizeBottom]="true"
      [resizeRight]="true"
      [minHeight]="50"
      [minWidth]="50">
     i am a common stream element
 </div>`

 `<div style="position:absolute"
       resizeBorder
       [resizeTop]="true"
       [resizeBottom]="true"
       [resizeLeft]="true"
       [resizeRight]="true"
       [minHeight]="50"
       [minWidth]="50">
      i am a absolute element
  </div>`
